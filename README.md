# 基于 STM32 的智能车载空气监控与联动系统

## 项目简介
本项目基于 STM32F4 开发，实现车载环境监测系统，支持温湿度、PM2.5、CO2等多传感器数据采集，并通过蓝牙进行无线传输，同时支持本地SD卡存储。

项目主要用于提升嵌入式系统开发能力，包括传感器驱动、多任务设计及通信模块实现。
---

## 功能特点
- 多传感器数据采集（温湿度 / PM2.5 / CO2）
- 蓝牙通信（BLE）
- 数据本地存储（SD卡）
- 实时监测与控制逻辑

## 软件设计

### 1. 数据采集
基于 I2C / UART / ADC 实现多传感器数据获取

### 2. 通信模块
基于 UART 实现 BLE 数据发送

### 3. 存储模块
通过 SD 卡进行数据记录

### 4. 系统控制
实现环境监测与基本控制逻辑
